⚙ Administrator setup
One time, ~2 minutes — this connects the dashboard so the owner can publish. (The client never needs this screen.) Paste a GitHub access token below.
How to get the token
  1. Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
  2. Repository access: Only select repositories → dinkclubwebsite.
  3. Permissions → Repository → Contents: Read and write.
  4. Generate, copy it, and paste it here. It's stored only in this browser.

No token yet? You can still edit and use Download file, then send us the file — but connecting lets you publish yourself.
